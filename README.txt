Wedding Photo Route Planner
===========================

This folder contains a self-contained wedding photo route planner.

Contents:
- index.html    : The main route planner (open in any browser)
- inspo/        : Folder containing all inspiration photos (35 images)

How to Use:
1. Open index.html in your web browser
2. Click spots to add/remove them from your route
3. Use the up/down arrows to reorder spots within each act
4. Click "More Ideas" items to add bonus photo concepts
5. Click any map marker to see details and Google Maps link

Sharing:
- Keep this entire folder together when sharing
- You can zip the folder to email it
- The selection state is saved in the HTML file

Route Features:
- Walking time calculated between each stop
- Segment-by-segment routing (no weird detours)
- Real-time end time estimation
- Budget tracking per act (40/100/100 minutes)
